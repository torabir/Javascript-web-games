# Descrition: 

4 classic small games intended for the web browser, created with JS, html and css. 

Games: 

- Frogger
- Whack-a-mole
- 2d-Breakout
- Space Invaders

# How to run: 

- Download all files from the folder of the game that you want to play.
- Open the html-file in a web-browser.
